# Lumina Library 📚✨

Lumina Library is a modern, responsive landing page designed for a premium self-study and research sanctuary. Built with a sleek, dark-themed user interface, it provides comprehensive information about study sections, operating hours, amenities, and community guidelines to offer visitors an optimal environment for academic excellence.

## ✨ Features

*   **Modern Dark UI:** High-contrast, visually appealing layout using gradient accents (purple/pink neon glows) engineered for ease of reading.
*   **Dynamic Stats Bar:** Displays key repository data at a glance (8 Study Sections, 12K+ Books & Resources, 500+ Study Seats, 6 AM Opening Time).
*   **Categorized Study Sections:** Filterable domains including Science & Technology, Arts & Literature, History & Geography, and Mathematics.
*   **Interactive Amenities Grid:** Showcases 8 essential premium facilities like High-Speed 1Gbps Wi-Fi, Silent Zones, Secure Lockers, and a Cafe & Snack Bar.
*   **Live Analog & Digital Clock:** Integrated real-time tracking display matching operational schedules.
*   **Community Guidelines Panel:** Clean layout detailing library decorum rules (Valid ID requirements, photography bans, luggage rules).
*   **Contact & Location Integration:** Full contact hub including address details, operating hours, pricing configurations, and a Google Maps toggle block.
